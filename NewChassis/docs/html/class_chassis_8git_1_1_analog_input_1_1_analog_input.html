Chassis.git.AnalogInput.AnalogInput.mode

This is the mode of operation for the analog inputs.

There are currently three modes available. Static mode is where one static voltage is acquired with no need for a sample clock. Finite mode is where a finite number of voltages will be acquired at a sample clock rate. Continuous mode is where a sequence of voltages are generated at a sample rate and then repeated until the stop() method is called.

Chassis.git.AnalogInput.AnalogInput.status

This is the status of the DAQmx task.

A value greater than 0 means that an error has occurred. When the status is greater than 0 an error should be reported by the class.
