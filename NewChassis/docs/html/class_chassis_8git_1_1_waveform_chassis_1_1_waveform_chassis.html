WaveformChassis Project
This project will utilized Python Code to generate analog and digital waveforms using a PXI Chassis and PXI-6733 modules. The analog and digital waveforms will have the capability of being synchronized using a NI-Sync card.

Detailed Description

This class contains a list of WaveformGenerator objects and a Timing object.

It is intended to represent a chasis with a NiSync Card and a number of PXI-6733s. It contains low level functions from the WCLowLevel class as well as high level functions.
