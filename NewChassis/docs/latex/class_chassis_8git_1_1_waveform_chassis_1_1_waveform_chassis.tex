\subsection{Detailed Description}
This class contains a list of Waveform\-Generator objects and a Timing object. 

It is intended to represent a chasis with a Ni\-Sync Card and a number of P\-X\-I-\/6733s. It contains low level functions from the \hyperlink{class_chassis_8git_1_1_waveform_chassis_1_1_w_c_low_level}{W\-C\-Low\-Level} class as well as high level functions.
