\subsection{Detailed Description}
This class contains a list of Waveform\-Generator objects and a Timing object. 

It is intended to represent a chasis with a Ni\-Sync Card and a number of P\-X\-I-\/6733s. It contains low level functions for the Wavefrom Chassis class.
